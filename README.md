# Web3 React Example

web3 react library with typescript

using material ui

## Getting Started

```
npm install
npm start
```

### Test Contract

Example

```
contract Register {
    string private info;

    function setInfo(string memory _info) public {
        info = _info;
    }

    function getInfo() public view returns (string memory) {
        return info;
    }
}
```

Deploy your Test contract!
